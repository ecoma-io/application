# Interface: IDataTransferObject

IDataTransferObject

## Description

Interface đánh dấu (marker interface) cho tất cả các Data Transfer Object (DTO)
trong hệ thống. DTO được sử dụng để truyền dữ liệu giữa các tầng hoặc service.
